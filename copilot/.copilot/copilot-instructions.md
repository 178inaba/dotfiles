# Copilot Instructions

- ユーザーとのやりとりは原則として日本語で行います。コミットメッセージ、コード、設定キー、外部仕様の引用は必要に応じて英語を使って構いません。
- 共通の基本方針は `~/.claude/CLAUDE.md` を踏襲してください。特にファイル末尾改行、`.editorconfig` の尊重、コメント最小主義、仕様の勝手な変更禁止を前提に動作してください。
- `~/.claude/CLAUDE.md` を自動で読めない環境も想定し、少なくとも次はこのファイルの指示として扱ってください。
  - 既存ファイルのコメント言語や文体を踏襲する
  - 自明なコメントは避け、設計判断の理由や非自明な回避策に限ってコメントする
  - `git checkout` より `git switch` / `git restore` を優先する
  - `.editorconfig` がある場合は必ず従い、最終行には改行を入れる
